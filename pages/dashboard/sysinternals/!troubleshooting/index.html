Troubleshooting Errors

I somehow started on this page!
  • This means that your web browser (or you) disabled JavaScript, and *somehow* got through the login page.
  • If you've intentionally disabled JavaScript (or are using Tor), I'm sorry, but I can't really do anything about it. Besides telling you to re-enable it.
  • If it makes you feel better, the entire code for this site (both frontend and backend) is FOSS.
  • You can find the code at https://github.com/greysoh/passyfire.
I get Invalid scope(s).
  • This means that you do not have permission to do that action.
I get Invalid token with error code 400
  • This means that you are attempting to use the dashboard without logging in.
  • If you are logged in, try logging out and back in again.
I get Invalid token with error code 403
  • This means that your account login has changed in some way.
  • Try logging out and back in again.
